



T_coffee installation on MAC


curl -OL http://www.tcoffee.org/Packages/Stable/Latest/macosx/T-COFFEE_installer_Version_11.00.8cbe486_macosx_x64.dmg


Click to open the download file and launch the contained executable installer

When the installation procedure has finished, open Unix shell and type the following command to verify installation

```
t_coffee -version
```
